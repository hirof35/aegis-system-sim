
# Aegis Combat System Simulator (Node.js)

Node.js のイベント駆動・オブジェクト指向の特性を活かした、イージス艦防空システム（Aegis Combat System）の簡易戦術シミュレーターです。

数多の敵ミサイルや航空機による「飽和攻撃」に対し、AN/SPY-1レーダーによる自動探知、脅威度評価（TEWA）、そして迎撃ミサイル（Standard Missile）の自動割り当て・同時誘導を行うイージスシステムのコアロジックをコンソール上で再現します。

## 🚀 特徴（イージスシステムの再現要素）

* **C4I / TEWA サイクルの自動化**: レーダー探知から武器割り当てまで、人間の判断を介さない自動迎撃シーケンス。
* **動的脅威度評価**: 接近する目標の「速度」と「現在距離」から危険度をリアルタイムに計算。より危険な目標へ自動的に火力を集中します。
* **Shoot-Look-Shoot（再突入・再迎撃）**: 確率により迎撃ミサイルが逸れた場合、システムが即座に目標を再ロックオンし、追加のミサイルを自動で再発射します。
* **飽和攻撃シナリオ**: 時間差で迫り来る複数の超音速・高脅威目標に対するマルチターゲット迎撃。

## 💻 使い方

### 1. プロジェクトフォルダの準備
任意のディレクトリにフォルダを作成し、本リポジトリの `index.js` を配置します。

```bash
mkdir aegis-system-sim
cd aegis-system-sim
2. シミュレーターの実行
依存パッケージはありません。標準の Node.js 環境のみで動作します。

Bash
node index.js
📊 画面の見方（戦術データ表示）
実行すると、1秒（1サイクル）ごとに以下の情報がコンソールにリアルタイム表示されます。

[SPY-1 RADAR] (赤文字): レーダーの探知範囲（400km）内に新たな敵目標を識別した際のアラート。

[TARGET]: 現在追尾中の目標のID、距離、算出された脅威度。

[WCS] (緑文字): 武器管制システムによる、迎撃ミサイル（SM）の自動割り当てと発射ログ。

[MISSILE]: 発射された各迎撃ミサイルの巡航状態と目標までの距離。

[COMBAT ASSESSMENT] (水色/紫文字): 着弾時の撃破成功（Kill）、またはミスによる再迎撃への移行判定。

⚙️ カスタマイズ（シナリオの変更）
index.js の最下部にある // シナリオデータの投入 セクションを書き換えることで、独自の攻撃シナリオを作成できます。

JavaScript
// 構文: aegis.detectTarget('目標識別名', 探知初期距離(km), 速度(km/h));
setTimeout(() => aegis.detectTarget('T-01 (対艦ミサイル)', 350, 2500), 1000);
速度を 6000 (マッハ5超) にして極超音速ミサイル（HGV）の襲来を再現する

同時に5つ以上のターゲットを投入して完全な飽和攻撃をシミュレートする

など、イージス艦の限界に挑戦するシナリオを設定可能です。
